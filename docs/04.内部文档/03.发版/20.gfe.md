# gfe（前端集成解决方案）
- 安装说明
- windows环境：  
1、安装nodejs，要求 v0.10.0及以上，v7.0以下  
2、安装gfe：npm install gfe -g  
3、进入npm/gfe安装目录默认路径：C:\Users\user\AppData\Roaming\npm\node_modules\gfe\，删除所有文件  
4、svn checkout以下地址到gfe下：https://code.ds.gome.com.cn/svn/atg_poc/30_Coding/gome-gfe/gfe  
5、进行cmd，执行gfe -v，如果出现版本号则说明你已安装成功  
   注：如需node高版本，可安装nvmw来控制node多版本切换    
   GFE维护的所有项目svn路径：https://code.ds.gome.com.cn/svn/atg_poc/30_Coding/NewDevMode/trunk/gome-gfe
