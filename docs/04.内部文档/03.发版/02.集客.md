## gscm里的地址
(http://gscm.ds.gome.com.cn/moduleControl.html) 项目路径为 atg_poc -> gome-member-marketing -> mms-front -> mms-frontplatform
![/img/mms/2.png](/img/mms/2.png)

因为用了npm run commit 所以不需要再去手动到gscm里去创建4位版本但是，如果新建了分支则需要修改新建分支的配置

![/img/mms/3.png](/img/mms/3.png)

* 如上图distSvn要换成新的分支路径
* branchesId也要换成新的分支的分支id
* 如果jenkins变了还需换Jenkinsurl不然会发版失败
