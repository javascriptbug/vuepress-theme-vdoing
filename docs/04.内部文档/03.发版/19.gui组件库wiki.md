# GUI组件库
1.	Svn地址：https://code.ds.gome.com.cn/svn/atg_poc/30_Coding/gome-gfe/gmlib
2.	访问地址：http://gui.ds.gome.com.cn或http://gui.gome.inc/  (域名现在不可以用，只能147的ip访问)
3.	服务器地址： 10.115.0.147
4.	LAMP环境目录：/opt/lampp
5.	数据库配置信息： /app/gmlib/config.php
6.	xampp安装步骤：
          
```
准备安装包：
xampp-linux-x64-5.5.28-0-installer.run
将项目copy到/app/gmlib下
(1)chmod 777 xampp-linux-x64-5.5.28-0-installer.run
(2)需要选项，一路Yes
(3)/opt/lampp/lampp start
    启动start服务器
(4)输入http:// 10.115.0.147访问不了，需要进行如下步骤：
  vi /opt/lampp/etc/extra/httpd-xampp.conf
   注释掉Require local
   #Require local
(5)重启xampp
   /opt/lampp/lampp restart
(6)此时http:// 10.115.0.147和http:// 10.115.0.147/phpmyadmin均能访问
(7)现在需要修改一下/opt/lampp/etc/httpd.conf文件，配置项目和静态化。
   DocumentRoot "/opt/lampp/htdocs"
   <Directory "/opt/lampp/htdocs">
   改为：
   DocumentRoot "/app/gmlib"
   <Directory "/app/gmlib">

   添加如下代码：
   Include "/opt/lampp/apache2/conf/httpd.conf"
<IfModule mod_rewrite.c>
RewriteEngine On
RewriteRule ^(.*)(/(list|page)(\-[0-9]+)+\.html)$ $1/index.php$2
RewriteRule ^(.*)(/tag\-.+?(\-[0-9]+)?\.html)$ $1/index.php$2
</IfModule>

```
7. 问题总结：   
     数据库改了密码后，phpmyadmin登陆不上去，需要以下步骤进行修改：
    
```
vi /opt/lampp/phpmyadmin/config.inc.php
```
找到$cfg['Servers'][$i]['password'] = '';重新设置mysql密码即可

8. 登录gui： 
- http://10.115.0.147/system/login.php 
- 账号：gfe/gomegfe

9. 服务挂了可以自己启动或者找部署启动
- 告诉他服务器地址 `10.115.0.147`，和启动命令`/opt/lampp/lampp start`
