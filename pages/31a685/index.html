# px2rem注意事项

  1. 直接写px，编译后会直接转化成rem —- 除开下面两种情况，其他长度用这个
  2. 在px后面添加/no/，不会转化px，会原样输出。 — 一般border需用这个
  3. 在px后面添加/px/,会根据dpr的不同，生成三套代码。—- 一般字体需用这个
.div{
    width: 150px;
    height: 64px; /*px*/
    font-size: 28px; /*px*/
    border: 1px solid #ddd; /*no*/
}

# wap端配置

https://prom.m.gome.com.cn/html/prodhtml/topics/202011/30/channeljcOVTNE2O4X.html

wap端需要在erm系统里配置，用的是林淋的促销模板

把新改的样式和js要先去erm里CMS配置管理-资源管理 的锅美优食里的js和css覆盖掉，如果重新换了名字，需要先把对应修改的css和js路径先替换到html/模板.txt里面，如果不更新，可能会有缓存缓存需要一定时间会自动清掉

把改好的代码合到html/模板.txt里面，再到erm里CMS配置管理-无线CMS-页面模板独立化配置，搜索条件key为 channeljcOVTNE2O4X，点编辑进入，把里面的内容用html/模板.txt里面的替换掉再提交

提交后再去erm里CMS配置管理-无线CMS-页面管理里，搜索条件key为 channeljcOVTNE2O4X，点编辑进入，再在右上角找发布
